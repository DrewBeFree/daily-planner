# Daily Planner

Multi-tab personal productivity app for daily tasks, grocery lists, and trip packing.

## Features

- **Tasks** — add, complete, and archive daily to-dos
- **Groceries** — create and manage multiple grocery lists
- **Trips** — packing lists per trip with archive
- Auto-print via Windows Task Scheduler each morning
- Supabase-backed persistence across devices

## Tech

HTML/CSS/JavaScript · Supabase SDK v2 · Auth module
